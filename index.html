Works with Claude Desktop

Connect PromptShelf to Claude Desktop using the Model Context Protocol (MCP). Access your entire prompt library without leaving your conversation.

  • List and search prompts
  • Get prompt content with versions
  • Filter by tags
  • Secure API key authentication
claude_desktop_config.json
{
  "mcpServers": {
    "promptshelf": {
      "url": "https://...cloudfunctions.net/mcp",
      "headers": {
        "Authorization": "Bearer YOUR_API_KEY"
      }
    }
  }
}
